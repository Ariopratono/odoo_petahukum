# Legal Regulations Module

Module untuk mengelola peraturan hukum dan perundang-undangan dengan lengkap.

## Fitur:
- Manajemen peraturan hukum lengkap
- Informasi detail peraturan (nomor, bentuk, tahun, dll)
- Status dan validitas peraturan
- Pencarian dan filter advanced
- Klasifikasi bidang hukum

## Field yang tersedia:
- Tipe Dokumen
- Judul
- T.E.U.
- Nomor
- Bentuk & Bentuk Singkat
- Tahun
- Tempat & Tanggal Penetapan
- Tanggal Pengundangan & Berlaku
- Sumber
- Subjek
- Status
- Bahasa
- Lokasi
- Bidang Hukum
